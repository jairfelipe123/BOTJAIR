## **Requisitos**

* Python 3.x (ultima versión si deseas)
*  INSTALAR Librerías en powershell o en CMD todo esta en `requirements.txt`

## **Compatibilidad**

Funciona en varios juegos FPS, **especialmente en Valorant**.
Debes asignar en el bot una tecla que **no haga una función muy importante dentro del juego**, para que pongas una tecla de disparo para que dispare automáticamente cuando detecte el color.

## **Recomendaciones**

1. **Ejecutar como Administrador** (lo principal).
2. Jugar en **modo ventana o ventana sin bordes**, porque algunos juegos bloquean el control del mouse en pantalla completa.
3. Si nada funciona, usa una **solución por hardware**:

   * Arduino Leonardo / Pro Micro (ATmega32U4).
   * El bot detecta el color → envía señal al Arduino → el Arduino hace el clic.
   * Para el juego es igual a un mouse real y no es detectable.
